# **Personal Website**

A blazing-fast, content-focused personal blog built with **[Astro](https://astro.build/)** and deployed globally using **[Cloudflare Pages](https://www.cloudflare.com/developer-platform/products/pages/)**.
This project combines modern web development practices with minimal JavaScript overhead and enhanced performance via edge rendering, animations, and selective hydration.

## **Technology Stack**

### **Core**

- **[Astro](https://astro.build/)**: Static site generation with partial hydration for components
- **[React](https://react.dev/)**: Component-based UI inside Astro pages
- **[Tailwind CSS](https://tailwindcss.com/)**: Utility-first styling with JIT compiler.
- **[Framer Motion](https://motion.dev/)**: Smooth, performant animations with physics-based transitions

### **Infrastructure**

- **[Cloudflare Pages](https://www.cloudflare.com/developer-platform/products/pages/)**: Global deployment with edge caching, custom domain and Git-based CI
- **[Cloudflare Web Analytics](https://www.cloudflare.com/web-analytics/)** – Privacy-focused, cookieless traffic metrics with zero performance overhead.
